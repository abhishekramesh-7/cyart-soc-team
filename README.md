It explains alert triage, risk scoring, and security frameworks like MITRE ATT&amp;CK, CVSS, and NIST SP 800-61 used in real SOC investigations.

This repository contains structured learning notes and practical concepts for Security Operations Center (SOC) fundamentals, including alert prioritization, incident classification, and the incident response lifecycle.

The project focuses on understanding how SOC analysts investigate and manage security events using industry frameworks and tools. It covers topics such as alert triage, risk scoring, CVSS-based prioritization, incident taxonomy, MITRE ATT&CK mapping, and the NIST incident response lifecycle.

The repository is designed as a learning and reference resource for aspiring SOC analysts and cybersecurity students, with explanations, real-world examples, and investigation workflows that reflect real SOC operations.

Key topics covered:

Alert Priority Levels and Risk Scoring

Incident Classification using security frameworks

Incident Response Lifecycle (Preparation → Lessons Learned)

SOC investigation workflows and methodologies

Frameworks and standards referenced include:

NIST SP 800-61 Incident Handling Guide

MITRE ATT&CK Framework

CVSS Vulnerability Scoring System

VERIS Incident Classification Model

This repository is part of my journey in learning SOC operations, threat detection, and incident response while building hands-on experience with SIEM tools and security monitoring platforms.
